>>>> Hướng dẫn chạy source code:
	1. Tại thư mục gốc chọn vào thư mục có tên là source.
	2. Nhấp chọn vào thư mục có tên là AV-Store.
	3. Tại thư mục AV-Store mở cmd bằng cách gõ cmd tại thanh đường dẫn.
	4. Gõ câu lệnh "cd api" và nhấn enter
	5. Chạy câu lệnh "npm install" để tải thư mục node_modules
	6. Sau đó để chạy API ta chạy lệnh "npm start" 
	7. Mở thêm một cmd nữa tại thư mục AV-Store
	8. Gõ câu lệnh "cd client" và nhấn enter
	9. Chạy câu lệnh "npm install" để tải thư mục node_modules
	10. Sau đó chạy lệnh "npm start" 
	11. Mở thêm một cmd nữa tại thư mục AV-Store
	12. Gõ câu lệnh "cd admin" và nhấn enter
	13. Chạy câu lệnh "npm install" để tải thư mục node_modules
	14. Sau đó chạy lệnh "npm start"
	+ Lưu ý: Khi chạy "npm start" vui lòng đợi một chút

>>>> Truy cập các website AV Store:
	+ Đối với trang phía khách hàng thì gõ đường dẫn: http://localhost:3000
	+ Đối với trang phía quản lý thì thay bằng: http://localhost:8080
